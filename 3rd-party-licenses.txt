This file is a list of the licenses of the third-party software used by
NetScanner and the other tools distributed with it. What follows is a
description of each third-party package. At the end of this file is the
license of each. The license of NetScanner itself is in the COPYING file.

NetScanner uses:
 o JH Software's DNS Client Library for .NET (v.1.0.1.0):
   Freeware Networking library, used for NSLookup.
   JHSoftware.DnsClient.dll file embedded into NetScanner.exe for portability.
   You can find more info here: http://www.simpledns.com/dns-client-lib.aspx
   If this page is not available, you can find it here:
   https://web.archive.org/search.php

 o Certain Nmap Scripting Engine scripts (NSE) use the simplified BSD license.

JH Software's DNS Client Library for .NET
=========================================
Copyright 2009 JH Software Aps

Files:
1) JHSoftware.DnsClient.dll
This is the library itself. 
Reference this file in your .NET project.
For ASP.NET projects place this file in the "Bin" folder.

2) JHSoftware.DnsClient.xml
This file provides intellisense in Visual Studio.
Place it in the same folder as the .dll file.
Not required/needed for production distribution.

3) JHSoftware.DnsClient.chm
This is the help file.
Not required/needed for production distribution.
